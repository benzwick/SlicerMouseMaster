Supported Mice
--------------

MouseMaster works with any mouse, but provides optimized profiles for:

.. list-table::
   :header-rows: 1

   * - Mouse
     - Vendor ID
     - Buttons
     - Status
   * - Logitech MX Master 3S
     - 0x46D
     - 7
     - Fully Supported
   * - Logitech MX Master 4
     - 0x46D
     - 7
     - Fully Supported
   * - Generic 3-Button
     - Any
     - 3
     - Basic Support
   * - Generic 5-Button
     - Any
     - 5
     - Basic Support

Don't see your mouse? Use the button detection wizard to create a custom profile.
